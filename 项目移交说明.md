## 核心功能

1. **商品价格监控**: 监控京东、淘宝商品价格变化
2. **历史价格记录**: SQLite 数据库存储价格历史
3. **Web 界面**: 展示商品列表、价格走势、监控状态
4. **自动化采集**: 通过浏览器驱动采集动态页面价格

## 数据库结构

表: `products`
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 |
| name | TEXT | 商品名称 |
| platform | TEXT | 平台(jd/taobao) |
| url | TEXT | 商品链接 |
| current_price | REAL | 当前价格 |
| lowest_price | REAL | 历史最低价 |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |

表: `price_history`
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 |
| product_id | INTEGER | 外键 |
| price | REAL | 价格 |
| recorded_at | DATETIME | 记录时间 |

---

## 注意事项

1. **Cookie 刷新**: 京东/淘宝登录态可能过期，需定期更新 cookies/ 目录下的 Cookie 文件
2. **网络环境**: 爬虫依赖网络访问，目标站点可能有反爬限制
3. **数据备份**: 重要数据定期备份 data/price-monitor.db
4. **端口占用**: 如 38472 端口被占用，修改 config.go 中的端口配置

## API 接口

| 接口 | 方法 | 说明 |
|------|------|------|
| /api/products | GET | 获取商品列表 |
| /api/products | POST | 添加商品 |
| /api/products/:id | DELETE | 删除商品 |
| /api/products/:id/price-history | GET | 价格历史 |
| /api/crawl | POST | 触发采集 |
